# Contributing to cloudsmith-msp

Thank you for your interest in contributing to CloudSmith!

## Developer Certificate of Origin (DCO)

All contributions must be signed off with the [DCO](https://developercertificate.org/).
Add `Signed-off-by: Your Name <email>` to every commit:

```
git commit -s -m "feat: my change"
```

## Conventional Commits

Commit messages must follow [Conventional Commits](https://www.conventionalcommits.org/):
`type(scope): description`

Types: `feat`, `fix`, `docs`, `chore`, `refactor`, `test`, `ci`

## Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes with DCO sign-off on each commit
4. Open a PR — the template will guide you through the checklist

## Questions

Open a GitHub Discussion or email the maintainers at maintainers@cloudsmith.cloud.
